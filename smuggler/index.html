The Stack
Foundation video models (open weights)
  • Wan 2.6 / 2.7 (Alibaba): current open workhorse; R2V learns motion, camera, style from a 5s reference clip; 2.7 adds deterministic composition with 9-grid I2V anchor frames
  • HunyuanVideo 1.5 (Tencent): 13B T2V/I2V, 720p at 24fps, upscales to 1080p, runs on 8 to 16 GB VRAM
  • LTX-2 (Lightricks): 19B DiT that generates synced video + audio in one pass; NVFP4 checkpoints push 4K locally
  • Nvidia Cosmos: 7B and 14B, strong physical-world prior, 1280x704 at 121 frames on a 12 GB card
  • Mochi 1 (Genmo): 10B AsymmDiT, best-in-class text adherence, 480p
Foundation video models (closed, via ComfyUI API nodes)
  • Veo 3 and Veo 3 Fast (Google): high realism, native audio, fast variant for iteration
  • Runway Gen-4.5: Elo leader at launch, pairs with Act-Two
  • Kling 2.1, MiniMax Hailuo-02, Vidu: side-by-side options for look variety
Control, consistency, character
  • VACE (Wan 2.1/2.2): unified T2V, R2V, V2V, inpainting, outpainting
  • Wan 2.6 Reference-to-Video: clone appearance + voice + motion from one clip
  • Wan 2.2 Animate: skeleton-based motion transfer + expression fidelity
  • UniAnimate: human animation with different quality profile to Wan Animate
  • One-to-All Animation 14B (CVPR 2026): alignment-free, integrated in Kijai WanVideoWrapper
  • BindWeave: character consistency when LoRA training is not an option
  • Character LoRA training: 15 to 50 images, 1000 to 3000 steps, for hero continuity
  • IP-Adapter / face embedding: fast previz consistency
  • Wan 2.2 Fun Camera: real camera language as Camera Control Codes (pan, tilt, dolly, orbit, crane, tracking)
  • Depth Anything V2 + ControlNet: spatial stability across cuts
Performance and acceleration
  • LightX2V V2 distillation LoRA: 40 steps down to 8 to 14, ~4x speedup
  • NVFP4 / NVFP8 checkpoints (lightx2v): ~60% less VRAM, up to 3x faster on Blackwell/Rubin
  • Kijai WanVideoWrapper: block swapping, FP8/GGUF quantization, torch.compile
Finishing: upscale and interpolation
  • SeedVR2 v2.5 (ByteDance): one-step video restoration, 60% less VRAM, GGUF quant, unlimited length
  • FlashVSR: competing open video super-res, commonly A/B tested against SeedVR2
  • Topaz Video AI: still best for archival, interlaced, VHS, film-grain sources
  • RIFE VFI (4.0 to 4.9): default frame interpolation, TensorRT variant for speed
  • GIMM-VFI: sharper intermediates, occasional artifacts
  • FILM: safest for big-motion scenes
  • GMFSS Fortuna: anime-optimized
Long-form continuity and stitching
  • ComfyUI-Wan-SVI2Pro-FLF (Well-Made, thinkWHY2046 forks): first-frame / last-frame keyframes, seamless stitching across 7+ clips
  • Stable Video Infinity 2.0 + Wan 2.2 I2V A14B: 5-pass pipeline handing motion latents clip to clip
  • ComfyUI-Wan-VACE-Prep (stuttlepress): interactive canvas for outpainting source frames
Dialogue, lip sync, audio
  • LatentSync 1.6 (ByteDance): 512x512 training, fixes teeth/mouth blur, 4K portrait output
  • MultiTalk (MeiGen-AI): multi-speaker with Uni3C camera stabilization, 480p to 720p
  • Stable Audio 2.5: 3-minute tracks in under 2s, audio inpainting, API node
  • ComfyUI_MusicTools, DJ_VideoAudioMixer: stem processing and A/V sync
  • HeartMuLa: open-source Suno-style generator inside ComfyUI
Image editing for frames, boards, inserts
  • Qwen Image Edit 2511: natural-language edits, up to 3 input images, compositing
  • Nano Banana Pro (Google Gemini): partner node, pairs with Kling Omni Pro and Wan 2.6
Performance capture and real-time
  • Runway Act-Two: webcam-to-character full-body + face + hands transfer
  • Krea Realtime 14B: 11 fps on B200, ~1s time to first frame, live prompt-surfing
  • Krea integrated suite: browser access to Veo 3, Kling, Hailuo, Wan, Runway; 8K upscale, 120 fps interpolation
3D and Gaussian splat previz
  • ComfyUI-3D-Pack (MrForExample): NeRF and 3DGS generation
  • ComfyUI-Sharp: single-image to 3DGS
  • GeometryPack + GaussianViewer: splat preview and PLY export
  • KHR_gaussian_splatting glTF 2.0 (Feb 2026): industry-standard splat interchange
Horizon tech (emerging, track closely)
  • Runway GWM-1 (Worlds, Avatars, Robotics): real-time interactive environment simulation
  • Google Project Genie / Genie 3: playable 3D worlds at 24 fps 720p, 60s sessions
  • Decart Oasis: commercial real-time world model
  • Streaming ComfyUI graph mode (expected 2026): rolling-buffer video generation
  • Wan 2.7 deterministic composition: physics-aware controllable diffusion
  • Splat-conditioned video: geometry as explicit spatial conditioning
  • Marker-free full-body capture: the direction Act-Two is pushing
